# 🌐 Chrome Extension – Leads Saver

This is a simple **Chrome Extension** that lets you save the current tab URL and store it in your browser using localStorage.

---

## 🚀 Features
- Save the current active tab
- Save any url by typing it manually and hiting the **SAVE INPUT** button
- Store links permanently (using localStorage)
- Display saved links in a list
- Lightweight and fast

---

## 🧩 How to Install (Enable in Chrome)

Since this extension is not published on the Chrome Web Store, you can run it manually:

### Step 1: Download the project
- Click on the green **Code** button on GitHub
- Select **Download ZIP**
- Extract the ZIP file

---

### Step 2: Open Chrome Extensions
- Open :chrome://extensions/
- Go to: top right corner
- Turn ON **Developer Mode**

---

### Step 4: Load the Extension
- Click on **Load unpacked**
- Select the extracted project folder

---

### Step 5: Done 🎉
- The extension will appear in your Chrome toolbar
- Click on it and start saving tabs!
